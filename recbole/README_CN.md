# recbole

RecBole 的顶层包，一个基于 PyTorch 构建的开源推荐系统框架。它为各种推荐范式提供了统一的数据处理、模型训练、评估和超参数调优接口。

## 文件

| 文件 | 描述 |
|------|------|
| `__init__.py` | 包初始化文件；定义版本号（`__version__`）。 |

## 子目录

| 目录 | 描述 |
|------|------|
| `config/` | 配置模块，用于从 YAML 文件、命令行参数和字典中加载和管理参数。 |
| `data/` | 数据处理包，包括数据集加载、交互表示、数据转换和数据加载器。 |
| `evaluator/` | 评估框架，提供基于排名和基于数值的指标（如 Hit、NDCG、MRR、AUC）。 |
| `model/` | 推荐模型实现，涵盖通用、序列、上下文感知、知识感知和外部库模型。 |
| `trainer/` | 训练编排模块，包括适用于各种模型类型的训练器和超参数调优。 |
| `sampler/` | 负采样策略，包括均匀分布、基于流行度和知识图谱的采样。 |
| `utils/` | 工具函数，包括日志记录、枚举类型、参数列表、随机种子初始化和 Weights & Biases 集成。 |
| `quick_start/` | 快速启动入口，提供 `run()` 和 `run_recbole()` 等高级 API 以便快速实验。 |
| `properties/` | 默认 YAML 配置文件，包含全局设置、各模型配置和数据集配置。 |
